On January 27, 2026, Moonshot AI quietly released Kimi K2.5 with a feature called **_Agent Swarm_**. A trainable orchestrator that could spin up 100 sub-agents and coordinate ~1,500 tool calls without a human touching anything.

Nine days later, on February 5, Anthropic shipped Claude Opus 4.6 alongside Agent Teams in Claude Code

> Agent teams are system where a lead agent delegates to teammates, but a developer still approves the plan.

Between those two dates, OpenAI launched the Codex desktop app with parallel agents on February 2. VS Code 1.109 shipped multi-agent support on February 5. Google Jules was already running 15 concurrent tasks. And Cognition’s Devin had started using Opus 4.6 under the hood.

> Four companies. Ten days. The entire shape of AI-assisted development changed.
