Forget single-agent workflows, Claude Code Agent Team is a new way to run agents in parallel coding.

> **Claude Code Agent Teams let multiple AI agents work together with shared tasks, direct messaging between teammates, and independent context windows.**

If you’ve been using Claude Code agents, you understand the flow: one agent, one task, sequential execution, everything flows back to the main session.

> **But with Claude Code Agent Teams, you can set up five AI agents to investigate a bug in your project, but instead of working independently, you will watch the five AI agents debate competing theories about the bug, and challenge each other’s assumptions.**

For example, if the task is to investigate a bug in my WebSocket implementation.

- _One agent proposes a theory about connection timeouts._
- _Another agent pushes back with evidence from the codebase that contradicts it._
- _A third agent ran tests that disproved the first theory but_…
