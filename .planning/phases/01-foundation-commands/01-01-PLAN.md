> **Correction (2026-03-05):** The command descriptions in Task 1 specify "(zero LLM tokens)" in the `description` frontmatter field. This claim is inaccurate for the Claude Code invocation path — `disable-model-invocation: true` prevents Claude from automatically invoking commands, but when a user invokes them, the model still processes the command markdown and output. The scripts themselves are deterministic (no LLM calls). The "zero LLM tokens" framing originated from the RLM research where REPL globals call script functions directly without model involvement. See CLI-01 in REQUIREMENTS.md for standalone CLI tracking.
